# DI Studio - Файлы сайта

## Основные страницы сайта:

### 1. index.html
- Главная страница сайта
- Современный дизайн с анимациями
- Герой-секция, услуги, портфолио, процесс работы, контакты
- Полностью адаптивная верстка
- Интеграция с Font Awesome иконками

### 2. about-page.html 
- Страница "О нас"
- История компании
- Команда специалистов
- Наши ценности и принципы работы
- Партнеры
- Временная шкала развития студии

### 3. services-page.html
- Детальное описание всех услуг
- Дизайн-проект, 3D-визуализация, проектная документация
- Авторский надзор
- Цены и сроки выполнения
- Этапы работы над проектом
- FAQ секция

### 4. drawings-page.html
- Страница чертежей и проектной документации  
- Типы чертежей (обмерный план, развертки, планы и т.д.)
- Преимущества работы с нами
- Форма заявки на проектную документацию
- Часто задаваемые вопросы

### 5. portfolio-demo-complete.html
- Полное портфолио работ студии
- Фильтрация проектов по категориям
- Галерея изображений
- Подробные описания проектов

### 6. DI Studio - Дизайн интерьера и проектирование.html
- Исходная версия главной страницы
- Содержит много контента и функций
- Калькулятор стоимости
- Отзывы клиентов
- Полная интеграция всех секций
